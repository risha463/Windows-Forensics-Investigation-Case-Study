# Windows-Forensics-Investigation-Case-Study

This project documents my investigation of a compromised Windows Server machine using Windows Forensics techniques.

The objective was to identify attacker activity, persistence mechanisms, malicious files, command and control infrastructure, and evidence of compromise.

Skills Practiced:

- Windows Event Log Analysis
- Scheduled Task Investigation
- Firewall Rule Analysis
- Hosts File Analysis
- Persistence Detection
- Malware Identification
- Command & Control Detection
- Digital Forensics
